# Simple Console Snake Game

A classic Snake game implemented in C++ for the console.

## Controls
- **W**: Move Up
- **A**: Move Left
- **S**: Move Down
- **D**: Move Right
- **X**: Exit game

## How to Compile and Run
Make sure you have g++ installed. Then run:

```bash
g++ main.cpp -o snake
./snake          # On Windows: snake.exe
